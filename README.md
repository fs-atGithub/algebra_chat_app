## Getting started

**Requirements**

- Node.js 18.3.0+
- npm 8.12.2+
- Scaledrone channel ID

**First steps**

1. Clone this repository
2. Run `npm install`
3. Run `npm start`

## Scripts

### `npm run start`

Start the development environment.

### `npm run build`

Build the project for production.

- [x] Send messages in real time
